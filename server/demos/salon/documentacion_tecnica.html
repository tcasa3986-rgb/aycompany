Guía de Instalación Local

Paso a paso para instalar el sistema en una computadora nueva

ℹ️

Requisitos del Sistema

Windows 10/11 (64-bit), macOS 12+ o Ubuntu 20.04+. Mínimo 4 GB RAM, 2 GB de espacio libre y conexión a internet.

📥 Programas a Instalar

  • A

    Node.js v18 LTS o superior

    Descarga desde: https://nodejs.org/es/download → Selecciona la versión LTS → Instala con opciones por defecto. Verifica:

    node --version
    npm --version
  • B

    MySQL Community Server 8.x

    Descarga desde: https://dev.mysql.com/downloads/mysql/ → Selecciona Developer Default. Anota la contraseña que pongas para el usuario root.

  • C

    Git

    Descarga desde: https://git-scm.com/downloads → Instala con opciones predeterminadas.

🚀 Configuración del Sistema

  • 1

    Clonar el repositorio

    git clone https://github.com/tu-usuario/sistema-salondebelleza.git
    cd sistema-salondebelleza
  • 2

    Instalar dependencias del Backend

    cd backend
    npm install
  • 3

    Crear el archivo de entorno Backend

    Crea el archivo backend/.env con:

    PORT=5000
    DB_HOST=localhost
    DB_USER=root
    DB_PASSWORD=tu_contraseña_mysql
    DB_NAME=salon_belleza_db
    JWT_SECRET=clave_secreta_muy_larga_y_segura
  • 4

    Crear la base de datos automáticamente

    node setup_db.js

    Crea todas las tablas y usuario admin: admin@salon.com / admin123

  • 5

    Instalar dependencias del Frontend y configurar

    cd ../frontend
    npm install

    Crea el archivo frontend/.env:

    VITE_API_URL=http://localhost:5000/api
  • 6

    Iniciar el sistema (dos terminales)

    Terminal 1 — Backend:

    cd backend && node server.js

    Terminal 2 — Frontend:

    cd frontend && npm run dev

    Accede en: http://localhost:5173

Despliegue en Railway + Vercel

Guía completa para publicar el sistema en internet

☁️ Fase A: Subir Código a GitHub

  • 1

    Crear repositorio en GitHub.com

    Ve a https://github.com → New Repository → Nombre: salon-belleza → Privado → Crear.

  • 2

    Subir el proyecto

    git init
    git add .
    git commit -m "Sistema Salón de Belleza v1.0"
    git remote add origin https://github.com/tu-usuario/salon-belleza.git
    git push -u origin main

🚂 Fase B: Backend en Railway

  • 3

    Crear cuenta y proyecto en Railway

    Ve a https://railway.app → Login with GitHub → New Project → Provision MySQL. Copia las credenciales de conexión generadas.

  • 4

    Añadir servicio Node.js

    En el proyecto: Add Service → GitHub Repo → Selecciona tu repo → Root Directory: /backend

  • 5

    Configurar Variables de Entorno en Railway

    PORT=5000
    DB_HOST=→ (valor de Railway MySQL)
    DB_USER=→ (valor de Railway MySQL)
    DB_PASSWORD=→ (valor de Railway MySQL)
    DB_NAME=→ (valor de Railway MySQL)
    JWT_SECRET=cadena_aleatoria_muy_segura_32chars
  • 6

    Inicializar DB y obtener URL pública

    Desde la consola de Railway: node setup_db.js
    En Settings → Networking → Generate Domain → Copia la URL: https://tu-app.railway.app

▲ Fase C: Frontend en Vercel

  • 7

    Crear cuenta y proyecto en Vercel

    Ve a https://vercel.com → Sign Up with GitHub → Add New Project → Selecciona tu repo → Root Directory: frontend → Framework: Vite

  • 8

    Añadir variable de entorno

    VITE_API_URL=https://tu-app.railway.app/api
  • 9

    ¡Deploy!

    Clic en Deploy. En 2-3 minutos el sistema estará en internet en la URL: https://salon-belleza.vercel.app
    Para dominio propio: Settings → Domains → Agrega tu dominio personalizado.

🌐

¡Sistema en Producción!

El sistema estará accesible desde cualquier computadora o celular con HTTPS automático. Cada push a GitHub redesplegará automáticamente.

⚠️

Seguridad Importante

Nunca subas el archivo .env a GitHub público. Usa variables de entorno en Railway y Vercel. Cambia el JWT_SECRET a una cadena aleatoria de mínimo 32 caracteres exclusiva de tu instalación.
